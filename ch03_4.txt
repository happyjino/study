ch03_4 - 객체의 비구조화 할당문
비구조화란?(destructuring)
구조화된 데이터를 분해하는 것!
비구조화 할당은 ESNext JS 구문이다. 그렇기 때문에 TS에서도 사용할 수 있다
비구조화 할당을 객체에 적용하려면 얻고 싶은 속성을 중괄호로 묶는다.
ex) let {name, age} = jack; // jack.name, jack.age를 얻는다.

잔여 연산자(rest operator)
... 연산자로 여러 개의 속성을 가지고 있을 때 몇 개를 제외한 나머지를 선택할 수 있다.
ex) let address: any = {
  country: 'Korea',
  city: 'Seoul',
  address1: 'Gangnam-gu',
  address2: 'Sinsa-dong 123-456',
  address3: '789 street, 2 Floor ABC building'
}

const {country, city, ...detail} = address;
console.log(detail);
출력 : 
{ address1: 'Gangnam-gu',
  address2: 'Sinsa-dong 123-456',
  address3: '789 street, 2 Floor ABC building'}

전개 연산자(spread operator)
객체의 속성을 전개하는데 사용한다. 나도 무슨 소린가 했는데 꺼낸다고 생각하면 될 것 같다!
비구조화 할당문이 아닌 곳에서 ...이 사용되면 전개 연산자라고 한다~
ex)
let part1 = {name: 'jane'}, part2 = {age: 22}, part3 = {city: 'Seoul', country: 'Kr'};
let merged = {...part1, ...part2, ...part3}
console.log(merged);
출력 : {name: 'jane', age: 22, city: 'Seoul', country: 'Kr'}
